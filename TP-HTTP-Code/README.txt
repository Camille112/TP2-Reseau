In order to launch the server, in a terminal open in TP-HTTP-Code\src :
$ javac http/server/*.java
$ java http.server.WebServer

Once it is open you can open http://localhost:3000/ and it will show the index.
To get a file you can write the file with its file extension following the url.
Example : http://localhost:3000/file.txt

You can use the other request HEAD, PUT, POST, DELETE in postman.
